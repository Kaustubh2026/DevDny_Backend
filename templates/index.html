How to Use Smart Event Planner
Creating Events

Fill out the form with event details and click "Create Event".

Checking Weather

Click "Check Weather" on any event to see the weather suitability.

Alternative Dates

If weather is poor, click "Get Alternative Dates" for suggestions.

Managing Events

Use the Edit and Delete buttons to manage your events.

Weather History

View past weather data for any event using the Weather History button.
